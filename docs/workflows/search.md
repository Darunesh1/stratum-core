# Search & Query Validation Checklist

Run these steps to prepare your search query:

- [ ] Select or initialize your project workspace.
- [ ] Configure date boundaries and document types.
- [ ] Upload reference dataset (or query spreadsheets).
- [ ] Extract TF-IDF key terms to identify relevant search keywords.
- [ ] Run validation tests on keyword syntax.
- [ ] Group search hits by OpenAlex topic IDs.
- [ ] Filter out outlying/irrelevant topic IDs.
- [ ] Verify that anchor DOIs are covered.
- [ ] Inspect a random sample of 50 papers to check precision.
